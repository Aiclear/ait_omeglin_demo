# ait_omeglin_demo

一个简单的在线聊天应用

## 项目介绍

该项目是一个在线视频文字聊天的应用

### 使用的技术

- 后端技术栈 `Javalin` + `Hibernate` + `SQLite`
- 前端技术栈 `Html` + `Css` + `JavaScript`

## 功能列表

- 用户注册
- 用户登录
- 在线匹配视频文字聊天(陌生人匹配方式)
- 聊天过程中加好友功能(需要AI实现)

## 实现功能需求描述

#### 聊天过程中加好友功能(需要AI实现)

- 功能描述：当用户通过在线聊天匹配到用户之后，可以通过消息框发送一条`@friend`的指令，然后作为按钮展示在对话框中
  ，在线聊天的用户可以通过点击按钮来确认是否从陌生人变为好友，并出现在自己的好友列表中，并且可以通过发送
  `#my-friends`消息
  来查看自己的好友列表
- 技术限制：前端 Javascript + Html + Css 后端 不可以新添加任何的依赖，使用现有框架完成该功能
- 后端要求说明：使用表`user_friends`保存好友关系
